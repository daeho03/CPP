초기 C++의 새로운 기능들
	전혀 사용하지 않는 기능들이 매우 많음
	현재에는 오직 소수의 기능만 알면 됨
		bool 데이터형(C++98, 모던 C++)
		참조(reference)
		개체지향 프로그래밍(object-oriented programming)
			보통 객체지향이라고 함
				object는 물체를 얘기하는거라 개체지향
Bool 데이터형
	true/false
값에 의한 호출
	값을 복사해옴
	원본에 영향 없음
 	java의 오브젝트는 참조에 의한 호출인데
		C/C++은 오브젝트도 값에 의한 호출이 가능
참조에 의한 호출
	메모리 주소를 복사해옴
	원본이 바뀜
참조(Reference)
	포인터를 사용하는 좀 더 안전한 방법
	java만큼 제한적이진 않음
	컴퓨터는 참조를 모름
		포인터와 참조는 같은 어셈블리어 명령어를 생성함
		내부적으로 포인터와 같음
		컴파일러는 참조를 "포인터"로 바꿔 줌
	별칭이다
		int num = 100;
		int& reference = num; 	// &num 과는 다름
	NULL이 될 수 없음
		int & reference = NULL; 	// error
	초기화 중에 반드시 선언되어야 함
		int& reference; 		//error
	참조하는 대상을 바꿀 수 없음
	함수 매개변수로서의 참조도 가능
		코드가 더 깔끔해 짐
		NULL값이 들어올 수 없어 NULL체크를 안해줘도 됨
		소유하지 않은 메모리 장소를 가리킬 수 없음
